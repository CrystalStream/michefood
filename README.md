# Development

1.- Clone the project: `git clone`

2.- Install all the dependencies: `npm install`

3.- Create your own `env` file: `cp .env.example .env`

4.- Run it: `npm start`

You'll have a development server running on `http://localhost:3000/`

# Test

To run the test:

```
npm test
```
